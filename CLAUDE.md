# my-denver-card-free

See [README.md](./README.md) for full project details: stack, key directories, venue data model, UI layout, and deployment.

## Important conventions

- Never use em dashes or `--` in any user-visible text. Use commas, colons, semicolons, or restructure.
- Always run `make generate` (or `templ generate`) after editing any `*.templ` file.
- Generated `*_templ.go` files are committed to the repo.
- The static site output is `docs/index.html`, served by GitHub Pages.
